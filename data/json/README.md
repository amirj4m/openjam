# JSON data files

For consumers who don't want to import a SQL dump.

| File | Purpose |
|---|---|
| `words_en.json` | English lemmas + senses *(coming in v0.1.0)* |
| `translations_fa.json` | Persian translations *(coming in v0.1.0)* |

Each file is a JSON array of objects mirroring the corresponding SQL table.
